# Simon Game on QUTyboard

Embedded Simon memory game implemented using QUTyboard microcontroller.

## Features
- Random sequence generation
- Real-time button input
- LED display output
- Sound feedback
- Game state management

## Tech Stack
- C
- Embedded systems
- PlatformIO
- QUTyboard microcontroller

## Project Structure
include/     -> header files  
src/         -> game logic implementation  
platformio.ini -> PlatformIO configuration
